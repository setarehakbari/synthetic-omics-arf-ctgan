# Synthetic Omics (ARF vs CTGAN) — Reproducible Evaluation Framework

This repository provides a reproducible evaluation pipeline for synthetic gene-expression generation in Major Depressive Disorder (MDD), comparing **Adversarial Random Forest (ARF)** and **Conditional Tabular GAN (CTGAN)**. The framework benchmarks synthetic data across multiple complementary criteria, including utility, fidelity, privacy, feature-importance alignment, dilution, and stability.

## Associated manuscript
**Title:** Balancing Utility and Privacy: Evaluation of Generative Models for Gene Expression Data with Application to Major Depressive Disorder 
**Authors:** Setareh Akbari, Jamie Li, Brett McKinney. 
**Target journal:** BMC Bioinformatics (formatting in progress)

## What’s in this repo
- **ARF evaluation** scripts
- **CTGAN evaluation** scripts
- Scripts to reproduce **figures** and **tables**
- A minimal **quantum–adversarial** robustness proof-of-concept 

## Evaluation dimensions (high level)
1. **Utility:** downstream predictive performance (RF OOB accuracy)
2. **Fidelity:** per-gene KS/Wilcoxon with BH-FDR
3. **Privacy:** domain-separator AUC (real vs synthetic)
4. **Feature-importance alignment:** RF permutation importance + NPDR overlap/Jaccard
5. **Dilution (novel):** rank-shift and drop rate of real Top-K features after augmentation
6. **Stability:** bootstrap Jaccard reproducibility of Top-K features

## Repository structure
- `ARF_eval.R` — ARF-based synthetic data evaluation
- `CTGAN_eval.R` — CTGAN-based synthetic data evaluation
- `make-figures.R` — figure generation scripts
- `Fig-Tab.R` — table generation scripts
- `final figures.R` — final assembly/export helpers (if used)
- `GAN.py` — (optional) supporting code for generative modeling / experiments

> Note: Output figures are typically written to a `figures/` directory and summary files to a `results/` directory (depending on your local configuration).
